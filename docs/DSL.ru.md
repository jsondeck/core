# DSL v0.1 Спецификация

Полная спецификация языка JsonDeck v0.1.

## Краткая справка

Основная структура DSL:

```json
{
  "jsondeck": "0.1",
  "id": "id-игры",
  "title": "Название игры",
  "table": { "width": 800, "height": 600, "camera": { "mode": "fixed" } },
  "zones": { "zone_id": { "type": "free_space", "layout": "free" } },
  "cardTypes": { "type_id": { "title": "Карта" } },
  "initialState": { "cards": [] },
  "rules": [{ "id": "r1", "on": "game.started", "then": [] }]
}
```

## Основные концепции

### Версия (jsondeck)

Строка `"0.1"` — текущая версия языка.

### Идентификаторы (id)

Уникальные строковые идентификаторы без пробелов.

### Переменные (variables)

Глобальные переменные игры типом number, string, или boolean.

### Зоны (zones)

Области на доске для размещения карт. Имеют тип (семантику) и layout (позиционирование).

### Типы карт (cardTypes)

Определяют внешний вид, свойства и теги карт.

### Начальное состояние (initialState)

Какие карты размещены в какие зоны при старте.

### Правила (rules)

Реагируют на события условиями и командами.

### События (events)

- Встроенные: game.started, card.clicked, и др.
- Пользовательские: custom.my_event

### Выражения (expressions)

Строки начинающиеся с `$` разрешаются как выражения: `$vars.counter`, `$source`.
